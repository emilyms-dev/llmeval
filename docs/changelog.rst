Changelog
=========

0.1.0 (2024-10-01)
-------------------

Initial release.

- YAML/JSON test suite schema with Pydantic v2 validation
- OpenAI and Anthropic model adapters
- Async test runner with configurable concurrency
- LLM-as-judge scoring with weighted rubrics
- SQLite storage with prefix-aware run lookup
- Rich-formatted CLI reporter and side-by-side diff viewer
- Typer CLI (``run``, ``show``, ``diff``, ``list``, ``serve``, ``version``)
- FastAPI dashboard backend with run trigger endpoint
- React 18 + Vite + Tailwind dashboard
- GitHub Actions ``ci.yml`` and ``eval.yml`` workflows
